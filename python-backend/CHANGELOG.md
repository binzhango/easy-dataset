# Changelog

All notable changes to the Easy Dataset Python package will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial Python package structure
- Core package architecture with framework-agnostic design
- Package configuration with pyproject.toml
- Development tools setup (pytest, ruff, black, mypy)
- CLI interface skeleton
- Example usage scripts
- Comprehensive documentation
- uv virtual environment support

### Changed
- Complete rewrite from JavaScript/TypeScript to Python
- Migration from Prisma to SQLAlchemy ORM
- Migration from Next.js API routes to FastAPI

### Planned
- Document processing implementation
- LLM provider integrations
- Database models and migrations
- FastAPI server implementation
- Desktop application with Electron + Python backend
- Complete test coverage
- Full documentation
